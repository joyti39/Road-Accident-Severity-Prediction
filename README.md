# Road Accident Severity Prediction

# Project Overview
This project focuses on predicting the severity of road accidents using machine learning
techniques based on the US Accidents dataset.

# Dataset
- Source: Kaggle
- Dataset: US Accidents (2016–2023)
- Sample Size: 80,000 records
- Target Variable: Severity (1–4)

# Methodology
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training & Evaluation

# Models Used
- Logistic Regression
- Decision Tree
- Random Forest
- XGBoost

# Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-score
- Confusion Matrix

# Tools & Libraries
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
- Google Colab
